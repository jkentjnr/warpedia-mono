# warpedia-mono

Foundational research, data model, infrastructure plan, and design system for **warpedia.org** — an open, primary-source-rich, encyclopedic resource for the human record of war. A war-specific peer to Wikipedia and the broader Wikimedia ecosystem.

**Status:** pre-build research and design phase. No implementation yet.

## Where to start

- **[`docs/README.md`](docs/README.md)** — index of every document in this repository: charter, research, data model, metadata, infrastructure, contribution flow, and the visual design system.
- **[`docs/charter.md`](docs/charter.md)** — what Warpedia is, what it is not, and the principles all other decisions must align to. Read this first.
- **[Live design system](https://jkentjnr.github.io/warpedia-mono/)** — rendered UI kit prototypes and token previews, served from GitHub Pages.
